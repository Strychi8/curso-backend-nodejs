Repositorio con el **material del curso de Backend con Node.js** dictado en **Talento Tech**.

En este repositorio encontrarás:

- 📚 Material teórico
- 💻 Ejemplos de código
- 🧪 Ejercicios prácticos

El objetivo del curso es que los estudiantes aprendan a **desarrollar APIs backend con Node.js**, comprendiendo también los fundamentos de **Internet, HTTP, arquitectura de software y seguridad web**.

---

# 🧠 Contenido del curso

Durante el curso se trabajan los siguientes conceptos fundamentales:

### 🌐 Internet y Web
- Arquitectura cliente-servidor
- Protocolo HTTP
- Direcciones IP
- NAT
- Cookies

### ⚙️ Backend con Node.js
- Node.js
- Express.js
- APIs REST
- Rutas
- Middleware
- Manejo de errores

### 🏗 Arquitectura de aplicaciones
- Controllers
- Services
- Models
- Routes
- Separación de responsabilidades

### 🔐 Seguridad
- HTTPS
- SSL / TLS
- Autenticación con JWT
- Access Tokens
- Refresh Tokens

# 🎯 Objetivos del curso

Al finalizar el curso los estudiantes podrán:

- Comprender cómo funciona Internet y HTTP
- Construir APIs REST con Node.js y Express
- Consumir APIs desde un frontend
- Implementar autenticación con JWT
- Comprender conceptos básicos de seguridad web
